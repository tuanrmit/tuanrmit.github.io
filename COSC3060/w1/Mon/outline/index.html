Bootstrap

A set of pre-built classes with profound CSS rules and JS to decorate and add functions to web pages

  • Setup Bootstrap

    To start, we just need to include the Bootstrap CSS and JS files just as we write link tag to include CSS files and script tag to include javascript files!

  • Bootstrap container and grid layout

    Container
    .container
    .container-{breakpoint} with breakpoint of sm (small), md (medium), lg (large), xl (extra large), xxl
    .container-fluid
    Grid layout
    .each row is 12-column width

  • Bootstrap spacing

    Just like margin and padding spaces that we have learned and used. For instance,
    m- for margin
    p- for padding
    mt- for margin-top
    ms- for margin-left
    mt-5 for size of 5 for margin-top
    m-auto for auto sizing of margin from 4 sides

  • Boostrap components

    A collection of pre-designed and pre-coded UI elements that web developers can use to quickly build responsive and consistent websites. For instance,
    Buttons, Forms, Lists
    Header, Footer, Navigation (Menu) bar
    Cards, Progress bar

Team formation

Expectation

In week 1, I will let people to have the freedom to choose their own teams (must be in the same workshop slot with you) (Your Team, Your Choice).

Each team just needs to fill in their information in a group registration spreadsheet, which must have 4 members.

For the teams which don't have enough team members or for the people who are struggled to find a team then we put you in any random team(s) to make sure every group should have 4 members.

From Week 1 to Week 7, members in a team will work together to finish the group project.

Hopefully, this arrangement is the best of both worlds!
