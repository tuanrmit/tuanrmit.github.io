Bootstrap Review

Bootstrap is CSS and JS framework helpful for designing responsive website.

Bootstrap makes classes with semantic names out of CSS basic properties.

To decorate an html element, we add bootstrap classes to it.

Layout

Breakpoints address the responsiveness of a webpage. A breakpoint represents width of a device.
A breakpoint is a screensize point below which an html element is of 100% of width and above which the html element's max-width is applied.
